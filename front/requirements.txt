Frontend requirements for manual setup
=====================================

Runtime:
- Node.js 20 LTS or newer
- npm 10 or newer

Dependencies are installed from package.json with:
- npm install

Defined dependencies:
- react ^18.3.1
- react-dom ^18.3.1
- @types/react ^18.3.3
- @types/react-dom ^18.3.0
- @vitejs/plugin-react ^4.3.1
- typescript ^5.5.4
- vite ^5.4.2
